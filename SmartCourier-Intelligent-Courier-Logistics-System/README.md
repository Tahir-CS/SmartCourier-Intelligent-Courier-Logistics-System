# SmartCourier-Intelligent-Courier-Logistics-System

A C++ project simulating an intelligent parcel sorting, routing, tracking, and courier operations system for logistics companies.

## Structure
- sorting/: Intelligent Parcel Sorting Module
- routing/: Parcel Routing Module
- tracking/: Parcel Tracking System
- operations/: Courier Operations Engine
- include/: Shared data structures and headers
- docs/: UML, architecture, flowcharts, and reports
